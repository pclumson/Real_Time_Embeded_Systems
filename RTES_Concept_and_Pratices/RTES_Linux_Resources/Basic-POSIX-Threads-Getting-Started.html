POSIX Threading for Real-Time Embedded Linux

POSIX Threads, or "Pthreads", are used extensively in this course. They can be learned by example and by studying the manual pages, but tutorials on POSIX threading are available from a number of sources, and will be quite helpful if you are new to this topic.

POSIX is a set of standards for portable operating systems (Wikipedia POSIX Overvew). POSIX threads have mostly been used for concurrency and parallel programming for applications, however, they can be used for Real-Time applications and systems by making use of the POSIX Real-Time extensions and careful use of them on embedded Linux systems.

  2. Gallmeister, Bill. POSIX. 4 Programmers Guide: Programming for the real world. " O'Reilly Media, Inc.", 1995. - available here and reviewed here
  3. Carnegie Mellong Pthreads Tutorials - found here
